[attr] - Represents elements with an attribute name of attr.
[attr=value] - exactly value
[attr~=value] - exactly value
[attr^=value] - prefixed by value
[attr$=value] - suffixed by value
[attr*=value] - value contains at least one occurrence of value within the string
